  4. def Where(equipment: Equipment, guid: PlanetSideGUID, continent: Zone): Option[ItemLocation]

    An exhaustive search of the provided zone is conducted in search of the target Equipment object and a token that qualifies the current location of the object in the zone is returned.

    An exhaustive search of the provided zone is conducted in search of the target Equipment object and a token that qualifies the current location of the object in the zone is returned. The following groups of objects are searched: the inventories of all players and all corpses, all vehicles trunks, the lockers of all players and corpses; and, if still not found, the ground is scoured too.

    equipment

    the target object

    guid

    that target object's globally unique identifier

    continent

    the zone whose objects to search

    returns

    a token that explains where the object is, if it is found in this zone; None is the token that is used to indicate not having been found

    LockerContainer
