  7. class VehicleSpawnControlRailJack extends VehicleSpawnControlBase

First, this step waits for the vehicle to be completely ready to accept the driver. Second, this step triggers the player to actually be moved into the driver seat. Finally, this step waits until the driver is properly in the driver seat. It has failure cases should the driver or the vehicle be in an incorrect state.
